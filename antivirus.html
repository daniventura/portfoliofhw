Instalación Malwarebytes - Windows

Vamos al sitio web de Malwarebytes y descargamos el paquete de instalación.

Al abrir el ejecutable podremos elegir entre comenzar la instalación o definir algunas opciones avanzadas, como nosotros haremos.

Podemos designar la ubicación de instalación, si queremos un acceso directo en el escritorio y el idioma.

Ahora tenemos que seleccionar si la instalación a nivel profecional o usuario, solo de diferencian en funciones de servidor y la monetización.

Malwarebytes nos brinda la posibilidad de tener una extensión del navegador para controlar la publicidad y rastreadores y optimizar el rendimiento.

El siguiente paso es la instalación del progamas según nuestros ajustes.

Al abrir el programa nos pide que compremos el programa o que validemos dicha compra, pero podemos clickar en quizás luego.

Para poder empezar la prueba gratuita nos piden nuestro correo.

Así luce el programa al iniciarse, podemos hacer análisis cuando decidamos o programar los rutinarios, además de configurar las funciones de en tiempo real.

Instalación Comodo - Ubuntu

Para descargar nuestro antivirus iremos al sitio web de Comodo.

Después de la descarga ejecutaremos el comando "sudo apt install [ruta de nuestro paquete]" para instalarlo.

Es posible que nos pida ejecutar el script /opt/COMODO/post_setup.sh para algunas configuraciones; lo haremos con el comando "sudo /opt/COMODO/post_setup.sh".

Tendremos que leer los acuerdos de licencia avanzando por la página pulsando enter, después empezará a correr el script.

Cuando la consola nos muestre el texto "COMODO Antivirus is successffully..." como aparece en la imagen, ya habremos completado la instalación.

Es importante que mantengamos la base de virus actualizada, podemos hacerlo desde la pestaña "antivirus" en "virus database update".
