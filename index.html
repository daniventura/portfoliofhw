Propósitos Generales

    Esta web en la que te encuentras será evaluada por dos asignaturas; por un lado Lenguajes de Marcas, que evaluará la forma, en este caso el código desarrollado, principalmente HTML y CSS. Los criterios son los siguientes:

  • Utiliza HTML dividiendo el contenido en las secciones adecuadas
  • Utiliza CSS básico
  • Utiliza CSS avanzado
  • Navegabilidad
  • Diseño adaptativo

    Por otro lado, bajo Fundamentos Hardware se evaluará el contenido, el portfolio de instalaciones de software, el cual tendrá que cumplir los siguientes criterios:

  • Software ofimático
  • Comparativa software ofimático
  • Software de recuperación de datos y mantenimiento del sistema
  • Instala software antivirus
  • Comparativa antivirus
  • Software portable
  • Comparativa software portable
  • Información adicional
