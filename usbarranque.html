Rufus

Un USB de arranque es cualquier memoria USB desde la cual podamos arrancar nuestro sistema, normalmente se usan para ejecutar una iso de instalación de un sistema operativo, pero tambien podemos ejecutar otro tipo de programas como clonezilla, o algún S.O. ligero que tengamos instalado en la memoria.

Rufus es un software que nos permite convertir un pendrive en un USB de arranque a partir de una imagen ISO, es de los más utilizados para esta función, sencillo y confiable. Podemos descargarlo desde la página web del proyecto.

Tras la descarga tendremos que ejecutar el archivo que obtendremos, y se nos abrirá la interfaz de la aplicación. Antes que nada tenemos que saber que en el proceso tendremos que formatear el dispositivo USB, por lo que perderemos todos los archivos que tengamos en la memoria extraible.

En la configuracion del programa podemos seleccionar el dispositivo que convertiremos en booteable, la imagen queremos que ejecute nuestro USB al usarlo, y dar un nombre al dispositivo. También podemos configurar aspectos más avanzados como el sistema de archivos del formateo o el esquema de partición de la memoria.

Cuando hayamos definido las propiedades de la unidad, solo tendremos que clicar en "empezar" y esperar a que se complete la barra de progreso, una vez acabe, nuestro USB de arranque estará listo para usar.

Ventoy

Ventoy es una herramienta que no solo nos permite tener varias ISO disponibles para ejecutar en el arranque del sistema, si no que también nos da la opción de seguir utilizando el dispositivo como una memoria extraible, pudiendo así tener tanto sistemas operativos listos para instalar, como aplicaciones o cualquier otro archivo que queramos para modificar o añadir a nuestros equipos recien instalados.

Podemos acceder al repositorio en github del proyecto para descargar la herramienta.

Con la descarga obtendremos un archivo comprimido, tras descomprimirlo ejecutaremos la aplicación llamada Ventoy2disk.

Solo tenemos que seleccionar el dispositivo en el que queremos instalar ventoy y darle a instalar, aunque es recomendable darle un sistema de particionado GPT para su correcto funcionamiento, para ello podemos seleccionarlo en el menu desplegable de "option".

Nuestro USB ahora se llamará "Ventoy", solo tenemos que copiar las imagenes iso que queramos a este dispositivo y ya podremos usarlas.

Al arrancar un sistema desde nuestro USB con ventoy instalado veremos este menu donde podemos seleccionar cualquier iso que tengamos almacenada en la memoria para ejecutarla.
